# Resting

Resting allows the party to recover HP, spell slots, and other limited resources.

## Short Rest
- Takes **1 hour**.
- Spend **Hit Dice** to recover HP.
- Some class abilities recharge.

## Long Rest
- Takes **8 hours**.
- Restore all HP.
- Recover up to **half your spent Hit Dice**.
- Regain spell slots and most class abilities.
- Can only benefit from **one Long Rest every 24 hours**.

## Interrupted Rest
- Fighting, casting spells, or other strenuous activity may interrupt a rest.
- If interrupted for too long, the rest must be started again.
